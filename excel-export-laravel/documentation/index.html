#Intro

MySQL Data to Excel Export - Laravel v 1.0.0

PhpSpreadsheet Documentation

MySQL Data to Excel Export - Laravel is simple web report export tool, you do can use MySQL Data to Excel Export - Laravel tool for excel file.It can be easily customized as like page setup, set page margins, center page, show/hide gridlines, worksheet name, font, font size, cells fill color, cells border color, text alignment vertical, text alignment horizontal and cells width easily changeable. Excel cells is set number formats and data validation. MySQL Data is showing rows group wise and header(rows and columns merge).You can protected and unprotected easily set on excel worksheet cells.

#Requirements

  • PHP version ^7.0
  • Laravel ^5.5
  • PHP extension php_zip enabled
  • PHP extension php_xml enabled
  • PHP extension php_gd2 enabled

#How to install in existing laravel project

To install in local existing laravel project. As like: laravel_test

Please follow the steps bellow:

Step-1:

  • Extract the archive file on your machine that you've downloaded from CodeCanyon.
  • Import database table (mysql-to-excel-laravel/demo_database/excelexport_laravel_db.sql) into existing database.

Step-2:

Use Composer to install PhpSpreadsheet into your project. As like: laravel_test

composer require phpoffice/phpspreadsheet

Step-3:

Open the (mysql-to-excel-laravel/routes/web.php) page any text editor.

Copy all route and paste into your project. as like: laravel_test/routes/web.php

Step-4:

Copy all blade(mysql-to-excel-laravel/resources/views/...blade.php) page and paste into your project. as like: laravel_test/resources/views

Step-5:

Copy (mysql-to-excel-laravel/app/ExcelExport.php) page and paste into your project. as like: laravel_test/app

Step-6:

Copy Controllers (mysql-to-excel-laravel/app/Http/Controllers/...php) page and paste into your project. as like: laravel_test/app/Http/Controllers

Step-7:

Copy (mysql-to-excel-laravel/public/assets, images and media) folder and paste into your project. as like: laravel_test/public

Step-8:

Open your browser and enter your url (http://localhost/laravel_test).
